# Hugging Face 참조 가이드

## 개요

IoEHub 프로젝트는 Hugging Face의 웹 인터페이스 디자인과 UX 패턴을 참조하여 개발되었습니다. 이 문서는 각 컴포넌트에서 Hugging Face 스타일을 어떻게 구현했는지 설명합니다.

## 디자인 시스템

### 색상 시스템

Hugging Face 스타일에서 참조한 주요 색상:

- 주 액센트 색상: 파란색 계열 (`#0969DA`, `#1A7F37`, `#3FB950`)
- 배경색: 흰색과 매우 밝은 회색 (`#FFFFFF`, `#F6F8FA`)
- 텍스트 색상: 진한 회색과 검정 (`#24292F`, `#57606A`)
- 강조 색상: 노란색 (`#FFC107`)

### 타이포그래피

- 기본 폰트: 본문에 시스템 폰트, 헤더에 약간 더 굵은 폰트 적용
- 헤더 계층 구조: 각 헤더 레벨에 일관된 크기와 가중치 적용
- 코드 폰트: 모노스페이스 폰트 사용 (데이터 표시 시)

### 공간 및 레이아웃

- 여백 시스템: 일관된 여백 적용 (TailwindCSS 간격 사용)
- 그리드 구조: 반응형 그리드 레이아웃 (주로 1, 2, 3, 4열 구조)
- 카드 레이아웃: 그림자가 약간 있는 둥근 모서리 카드

## 주요 UI 컴포넌트

### 헤더 (Header.tsx)

Hugging Face 참조 요소:
- 로고 + 네비게이션 바 레이아웃
- 검색창 디자인 (돋보기 아이콘 포함)
- 모델, 데이터셋, 스페이스 등 주요 네비게이션 항목
- 로그인/가입 버튼 스타일

구현 세부 사항:
- 반응형 디자인으로 모바일에서는 햄버거 메뉴로 변환
- 검색창은 키워드 입력 시 자동 완성 제공 (추후 구현)
- 스크롤 시 상단 고정 (sticky) 헤더

### 트렌딩 섹션 (TrendingSection.tsx)

Hugging Face 참조 요소:
- 3열 그리드 레이아웃 (모델, 데이터셋, 스페이스)
- 인기 항목 카드 디자인
- 메타 정보 표시 방식 (다운로드 수, 업데이트 날짜)
- 카테고리 배지 스타일

구현 세부 사항:
- 각 카드에 호버 효과 추가
- 다양한 카테고리별 아이콘 및 색상 코딩
- 모바일에서는 단일 열로 변환

### 히어로 섹션 (HeroSection.tsx)

Hugging Face 참조 요소:
- 그라데이션 배경
- 중앙 정렬된 타이틀과 설명
- 상위 CTAs (Call-to-Action) 버튼
- 특징 그리드

구현 세부 사항:
- IoEHub 특화 그라데이션 (파란색-보라색)
- 반응형 그리드로 모바일에서는 단일 열로 변환
- 애니메이션 효과 (선택적용)

### 마켓플레이스 섹션 (MarketplaceSection.tsx)

Hugging Face 참조 요소:
- 카드 그리드 레이아웃
- 제품 메타 정보 표시 방식
- 호버 및 인터랙션 효과

IoEHub 특화 개선사항:
- IoT 디바이스에 최적화된 카테고리
- 제품별 아이콘 및 색상 코딩
- 별점 및 리뷰 시스템

### 푸터 (Footer.tsx)

Hugging Face 참조 요소:
- 멀티 컬럼 링크 그룹
- 소셜 미디어 아이콘
- 저작권 및 법적 링크

구현 세부 사항:
- 반응형 레이아웃
- 다양한 리소스 및 지원 링크 제공
- 회사 정보 섹션

## UI 패턴 및 상호작용

### 카드 컴포넌트

- 일관된 카드 디자인: 가벼운 테두리, 부드러운 그림자, 둥근 모서리
- 호버 상태: 그림자 증가 또는 테두리 색상 변경
- 배지 및 라벨: 상태 표시를 위한 작은 컬러 배지

### 네비게이션

- 상단 주 네비게이션: 주요 섹션 접근
- 탭 인터페이스: 관련 콘텐츠 그룹화
- 빵 부스러기 내비게이션: 깊은 계층 구조 이동 (추후 구현)

### 버튼 및 액션

- 주 액션: 채우기 버튼 (파란색 배경)
- 보조 액션: 아웃라인 또는 고스트 버튼
- 그룹화된 액션: 관련 버튼을 논리적으로 그룹화

## 애니메이션 및 전환

- 페이지 전환: 부드러운 페이드 인/아웃
- 요소 호버: 미묘한 변환 또는 색상 변경
- 로딩 상태: 스켈레톤 로더 또는 스피너 (컨텍스트에 따라)
